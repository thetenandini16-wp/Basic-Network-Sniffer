
Basic Network Sniffer

🔗 Project Description

The Basic Network Sniffer is a Python-based cybersecurity project that captures and analyzes live network traffic. It displays useful information such as source IP address, destination IP address, protocol type, and packet details. This project helps understand how network communication works and is developed for educational purposes.

🔗 Features
- Capture live network packets
- Display Source and Destination IP addresses
- Identify network protocols (TCP, UDP, ICMP, etc.)
- Analyze packet information in real time
- Simple and easy-to-use Python implementation

🔗 Technologies Used
- Python 3
- Scapy
- Socket Programming (if applicable)

🔗Requirements
- Python 3.x
- Scapy Library
- Administrator/Root Privileges
- Internet or Local Network Connection
-  pip install scapy
   

🔗 How to Run :
   
   • python network_sniffer.py
